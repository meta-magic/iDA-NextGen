User can select the option to generate the Live Report, as per the need. These reports are as follows:

  • Live Report: Reports generated by establishing connection with device and fetching the information directly from the device.
  • Schedule Report:Report will be generated on configured scheduled time. Devices for scheduled reports will be selected based on filtered criteria if applicable or else all devices will be selected. If devices are selected, report will be executed on the selected devices only.
