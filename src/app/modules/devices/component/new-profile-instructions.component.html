Instruction
  • Profile Name: Provide the name of the profile
  • Default Waiting Condition: Provide default waiting condition like ">,#". Here 2 conditions with comma-separated is expected. The sequence of the waiting condition is important.
    • (a)
    How to identify the first waiting condition: To identify, simply login to the device and then the last character of the output prompt is your first default character. In the following example, '>' is the first default character.
    Device Console:
    login as: cisco
    Using keyboard-interactive authentication.
    Password:
    WS-C2960X-24PD-L-01>
    (b)
    How to identify the second waiting condition: After login to the device execute enable command on the device then the last character of the output prompt is a second waiting condition. If enable command is not configured on the device, then execute configuration terminal command on the device and get the last character for second waiting condition. In the following example '#' is the second default waiting condition.
    Device Console:
    login as: cisco
    Using keyboard-interactive authentication.
    Password:
    WS-C2960X-24PD-L-01>
    WS-C2960X-24PD-L-01>enable
    Password:
    WS-C2960X-24PD-L-01#
  • Enable Command: Provide command to enter into privilege EXEC mode.
  • Running Configuration Command: Provide the command to fetch the current running configuration of the device.
  • Write Memory Command: Provide the command to save the running configuration onto the memory.
  • Terminal Length Command: Provide the command to disable pagination on the device for command output.
  • Configuration Terminal Command: Provide the command to enter into configuration mode.
